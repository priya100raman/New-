1 Introduction

This document describes the working of a web application. It describes briefly the components involved in developing a web apllication and how they interact with each other.

2 Web Server: Server

A web application runs in a seperate thread on a Web server. This application servers the data requested by the clients by pulling it from the datadase.

3 Browser : Client

A browser runs the client side of the application and sens request to the server as and when it needs data. Typically the client is the interface for a user to interact with the web application.

4 Database

Database persist the data that has been created/modified/updated by the client.

5 Communication betweem Server and Client

The web application exposes a sert of REST APIs that a client side of the application can use to communication with the server.
